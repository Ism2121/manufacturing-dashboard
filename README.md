# manufacturing-dashboard
A dashboard designed to provide insight on manufacturing efficiency.

The goal of this project is to gain a foundational understanding of manufacturing metrics and effeciency. It's a starting point for my research and develpoment of a service tool designed for wood manufacturing shops.

# Tools Used
sql/ excel/ tableau

# Dashboard
The dashboard will show:
 - cycle times
 - oce report
 - effeciency by part number
 - downtime / bottlenecks
